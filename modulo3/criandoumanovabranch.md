### Criando uma nova branch pelo terminal

Comando necessário para a criação de uma nova branch via terminal:


git checkout -b (nome da nova branch)

Este comando cria e já troca para uma nova branch

Para criar uma branch pelo Github:

1- Acessar o repositório 
2- Acessar a sessão de listagem das branches
3- Inserir o nome da nova branch
4- Confirmar a criação da nova branch

Caso seja necessário excluir uma branch pelo Github:

1- Acessar a sessão de listagem de branches
2- Clicar no ícone da lixeira para realizar a exclusão da branch
